# Покупка VPS с VPN

В этой статье подробно описано как приобрести и использовать сервер на [play2go.cloud](https://p2g.one/wiki) с предустановленным VPN 

:::info Обратите внимание
Для личного использования вместо тарифов линейки PROMO мы **рекомендуем** рассмотреть тарифы LC/RYZEN, так как они имеют **более высокую скорость сети**
:::

## Использование AmneziaWG {#awg}

При покупке виртуального сервера выберите дополнительное ПО `VPN: AmneziaWG`

![AWG](/vpn/select-awg.png)

Оплатите сервер, дождитесь активации и установки VPN
<br/>Как только сервер активируется и VPN установится, скопируйте конфиг AmneziaWG из панели управления виртуальным сервером

![AWG](/vpn/copy-awg.png)

Скачайте клиент AmneziaVPN c [официального сайта](https://amnezia.org/ru/downloads), установите и вставьте туда скопированный конфиг и нажмите `Подключиться`

![AWG](/vpn/awg.png)


Готово!

## Использование Xray (vless) {#xray}

При покупке виртуального сервера выберите дополнительное ПО `VPN: Xray`

![Xray](/vpn/select-xray.png)

Оплатите сервер, дождитесь активации и установки VPN
<br/>Как только сервер активируется и Xray установится, скопируйте конфиг vless из панели управления виртуальным сервером

![Xray](/vpn/copy-xray.png)

Установите Happ c [официального сайта](https://www.happ.su/main/ru), далее вставьте скопированный конфиг и нажмите на кнопку для подключения

![happ](/vpn/happ.png)

Готово!
